## Command Template

# 월드맵 메뉴·조작방법 텍스트 한글화

아래 텍스트들을 한글화해줘.

## 사용자 관찰 (렌더 경로 힌트)
- 이 글자들은 **타일 폰트에서 간단히 호출해서 불러오는 것으로 보임**. 글자들이 구성된 **타일 이미지(CHR)**를 확인함.
- 다만 **일반 타일맵/스프라이트에는 해당 문자가 그대로 뜨지 않음** → 간접 렌더로 추정.
- **파츠 화면에서 쓰던 "작은 폰트"를 활용해 번역**해달라(옵션부품 등 SJIS 8pt 계열일 가능성 — 먼저 규명).

## 먼저 할 일 (규명)
1. **세션 시작 리딩**: `tasks/next-session.md`·`tasks/lessons.md`·`docs/12`(SJIS)·`docs/16`(RE MCP)·`assets/translation_guide/glossary.md`·메모리 전체(특히 [[sjis-0x86-expansion-suspect]]·[[npc-field-dialogue-gap]]).
2. **렌더/폰트 경로 규명**: 월드맵에서 X버튼 메뉴가 **어느 폰트·어느 렌더 시스템**을 타는지 확정.
   - 후보: ① SJIS(System②, $D9 8pt, `build_sjis`/`sjis_ui.json`) — 파츠 옵션부품과 같은 경로면 가장 쉬움 ② 압축글리프(System①, $CA) ③ 별도 소형 타일 폰트.
   - 도구: `scripts/lua/trace_field_src.lua`(파서/씬표/디코더 동시 후킹, 화면 P/S/D 카운터) + 필요시 그래픽/VRAM 트레이스. **역디코드 기법**(깨진 한글→글리프인덱스→원문)도 활용.
   - "파츠에서 썼던 작은 폰트"가 무엇인지 특정(예: SJIS $D9 8pt면 `sjis_ui.json` 티어 추가로 해결).
3. **번역·재삽입**: 규명된 시스템의 파이프라인으로. SJIS면 `sjis_ui.json` 티어 추가(전각 표기·전각공백·슬롯/용량 확인 — 0x86 확장 반영됨, VRAM 224 상한 유의). 압축글리프면 `dialogue.json`+포인터.

## 2. 아이템 → 조작방법 메뉴 텍스트 (번역 필요 — glossary 준수)
> 규칙: **2글자 스탯 라벨(SP/CN/PW/DF/WT/DP/BP)은 원문 유지**(glossary), 설명만 번역. 전각 문장부호. 용어는 glossary(§2 머신·§4 일반용어: セッティング=세팅·パーツ=파츠·マシン=머신·グリッド=그리드·コース=코스 등) 앵커.

**もくじ(목차)**
- P2 - マップ モード
- P3 - セッティング モード
- P4 - パラメータについて
- P5 - グリッドへんこう モード
- P6 - レース モード
- P7 - WGPエントリー モード

**セッティング モード**
- マシンきりかえ / パーツせんたく / マシンきりかえ / ウィンドウ / けってい / キャンセル

**パラメータについて**
- SP スピードせいのう / CN コーナリングせいのう / PW パワーせいのう / DF ダウンフォースせいのう / WT おもさ / DP たいきゅうりょく / BP ばくそうポイント

**グリッドへんこう モード**
- マシンせんたく / マニュアル / しゅうりょう / けってい / キャンセル

**レース モード**
- チェンジ / ステアリング / ポーズ / チェンジ / トップチェンジ / BPかいふく / ばくそう / レースアイテム

**WGPエントリー モード**
- せんたく / けってい / キャンセル
