# 미니욘쿠 렛츠&고!! POWER WGP2 한글 패치 v0.9

## 적용 대상과 방법

이 패치는 아래의 헤더리스 2MiB 일본판 ROM에만 적용한다.

| 항목 | 값 |
|---|---|
| CRC32 | `4459D4D0` |
| MD5 | `acdeb2ee6ef7b460c5dfed6957f8581a` |

```bash
xdelta3 -d -s "원본.smc" "wgp2_kr_v0.9.xdelta" "wgp2_kr_v0.9.smc"
```

배포 패치의 크기는 201,332바이트이며 SHA-256은
`9842f14a0cb8a608f17b453f746a64bc65c659534c274a6ab4402e21d144f53a`이다.
역적용 결과가 최종 통합 ROM과 바이트 단위로 일치하는지 검증했다.

원본 ROM과 패치 적용 ROM은 배포 파일에 포함되지 않는다.

## 작업 내용

- 정적 대사 681개를 추출·번역·재삽입하고 전량 라운드트립을 검증했다.
- 어드벤처 스토리 VM과 압축 코덱을 역공학해 235씬·1,782메시지를 반영했다. 분기·목록 뒤에 숨은 대사 58개도 재감사로 복구했다.
- 장소마다 다른 압축 팩 안의 필드/NPC 텍스트를 전수 조사했다. 숨은 레코드 1,207개, 발생 텍스트런 1,411개, 고유 대사 1,340개를 위치보존 방식으로 재삽입했다.
- 월드맵 퀴즈 70문항의 질문·선택지 350문자열과 정답, 상태 표시, 오답 메시지를 한글화했다.
- 시작·저장 메뉴, 레이서·팀·머신·파츠·행성·옵션 부품 등 SJIS 메뉴/UI를 한글화하고 글리프 슬롯을 확장했다.
- 월드맵 X 메뉴, 조작방법 튜토리얼, 용어집, 지도, 이지·수동 세팅, 확인창, 다음 레벨 표시 등 소형 타일폰트 UI를 한글화했다.
- 타이틀, 부팅 크레딧, 일시정지 메뉴, 포메이션, 기기 능력치, 개러지 분류, 경기 HUD, 경기장명 8종, 챕터 인트로 10종, VICTORYS 에피소드 인터미션 그래픽을 한글화했다.
- Result 화면의 선수 ID 범위표 110개와 표시 이름 61종을 반영했다. 아돌프–FOX1 공유 타일을 분리하고 모든 표시 이름의 좌측 정렬을 통일했다.
- 파츠 획득 시 동적으로 조합되는 이름 조각과 레벨업·레이스 특수 메시지 등 복구 과정에서 빠졌던 텍스트를 다시 반영했다.
- 실제 엔딩 크레딧·베스트타임 45행을 전용 8×8 글꼴로 한글화했다. 런타임 레이스 기록 필드를 보존했으며, 총괄 프로듀서 뒤에 현지화 메시지 12행을 추가했다.
- 원본과 같은 2MiB ROM 크기를 유지했다. 스크립트 길이·VM 오프셋·포인터·그래픽 타일 영역을 회귀 검사해 월드맵 타일, 스프라이트, 메뉴와 스토리 진행의 손상을 막았다.
