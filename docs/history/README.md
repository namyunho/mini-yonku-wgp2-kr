# 역사 기록

현재 통합 빌드가 더 이상 채택하지 않는 설계, 병합 전 하위 브랜치 구현, 실패 실험의 근거를
보존한다. 이 문서들은 회귀 원인을 추적할 때 유용하지만 현재 구현의 정본은 아니다.

- `menu4-context-font.md` — 월드맵·튜토리얼·용어집·지도 문맥 글꼴의 최초 안정화 기록.
- `manual-setting-xmenu-handoff.md` — 수동 세팅 공유 글꼴 실험과 초기 인계.

현재 동작은 상위 디렉터리의 `README.md`, `18-menu-tile-font-labels.md`, 최신 빌드 스크립트를
우선한다.
