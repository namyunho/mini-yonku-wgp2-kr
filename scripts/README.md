# 스크립트 디렉터리

`build_all.py`가 배포 ROM을 만드는 유일한 통합 진입점이다. Python 파일은 서로 직접 import하고
문서·명령줄에서 파일명을 진입점으로 사용하므로, 패키지 하위 폴더로 옮기지 않고 접두사로
역할을 구분한다.

| 이름 | 역할 |
|---|---|
| `build_*.py` | 시스템별 재삽입·통합 빌드 |
| `extract_*.py` | ROM 데이터 추출·카탈로그 생성 |
| `export_*.py` | 수작업용 그래픽 템플릿 생성 |
| `test_*`, `verify_*`, `validate_*`, `audit_*` | 회귀·무결성·화면 범위 검사 |
| 그 밖의 분석 스크립트 | 주소 탐색, 디코드, 보고서 생성 |
| `lua/` | Mesen2에서 실행하는 캡처·조사·트레이스 |

경로를 추가하거나 변경하면 `build_all.py`, 관련 JSON 매니페스트, 문서의 재현 명령을 함께
갱신하고 전체 빌드의 크기와 해시를 비교한다.
