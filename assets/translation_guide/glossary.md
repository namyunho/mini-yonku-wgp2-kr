# 번역 용어집 — 고유명사·용어 통일 (SSOT)

원작: 「폭주형제 렛츠&고!! WGP」(爆走兄弟レッツ&ゴー!!WGP). 게임 「ミニ四駆 レッツ&ゴー!! POWER WGP2」 한글 패치용.
출처: 나무위키 「등장인물」·「등장 머신」(assets/translation_guide/*.mhtml)에서 갈무리 + 원어 확인.

## 명명 정책 (확정)

- **1차 버전 = 일본 원어 음차 기준** (세이바 고 등). 아래 표의 **「원어 음차」 열이 실제 적용값**.
- 한국 방영판(「우리는 챔피언」) 로컬명(남궁 호 등)은 **병기·학습만** → **차후 별도 "한국명 버전"** 제작용.
- **MAX(애니 3기)·Return Racers 계열은 이 게임(WGP2) 범위 밖 → 미사용**(§7 참고용).
- 머신명은 원작·방영판 모두 가타카나 음차 유지 → 정책 무관 **음차 통일**.

### 호칭 규칙 (중요)

- **원문이 이름만 부르는 호격은 그 형태를 그대로 살린다.** 성을 임의로 붙이지 말 것.
  - `ゴー！` / `豪！` → **「고!」** (❌ 「세이바 고!」 아님)
  - `レツ！` / `烈！` → **「레츠!」**, `リョウ！` → **「료!」**, `トウキチ！` → **「토우키치!」**
- 즉 대화 중 부르는 톤·친밀도를 보존 — 풀네임(세이바 고)은 **원문에 풀네임이 나올 때만** 사용.
- 별명·애칭도 동일 원칙: 원문이 애칭이면 애칭 그대로(예: `おおがみ` 없이 부르면 그대로).

## 4. 일반 용어 통일 (UI/대사 공통, 확정)

| 원문 | 한국어 | 원문 | 한국어 |
|---|---|---|---|
| マシン | **머신** | 決定 / もどる | **결정 / 뒤로** |
| ガレージ | **개러지** | 中止 / 続ける | **취소 / 계속** |
| セッティング | **세팅** | セーブ / ロード | **저장 / 불러오기** |
| フォーメーション | **포메이션** | パーツ | **파츠** |
| グリッド | **그리드** | コース | **코스** |
| エントリー | **출전**(문맥) | ミニ四駆 | **미니사구**(차종 총칭) |

- 조작 안내: `Aボタンで決定　Bボタンでもどる` → **「A버튼 결정　B버튼 뒤로」**(전각공백 구분 유지, 라틴 A/B 유지).
- 전각 물음표·느낌표(？！)·선택지 들여쓰기(전각공백) 원문 위치 존중.

### 줄 길이 규칙 (중요) — 원문 기준

- **기준 = 원문.** 각 줄은 **해당 대사 원문의 줄 길이에 준해** 채운다. 원문이 쓰는 폭만큼 쓰되, **공간을 억지로 남기지 말 것**(너무 짧게 줄이지 않는다).
- **상한**: 그 대사 원문의 최장 줄 길이 이내, **전역 ≤ 16 한글 음절**(공백·문장부호 제외 카운트).
- 근거(실측): 원문 줄당 최대 ≈ **17~18자**(전각 일본어; 대사「/튜토리얼『/상태（/외침 전 유형 16~19자). 별도 좁은 맵 대화창 없음(거의 전체 폭 사용). 게임은 **VWF**라 한글이 가나보다 폭이 약간 넓어 원문 최대보다 1~2자 여유를 둔 **16 음절**을 전역 상한으로.
- 넘칠 때만 의미 유지하며 `{nl}` 추가/축약. 최종 폭은 Mesen2 실측으로 확인(near-cap 라인 플래그).
