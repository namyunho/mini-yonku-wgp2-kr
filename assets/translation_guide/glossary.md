원작: 「폭주형제 렛츠&고!! WGP」(爆走兄弟レッツ&ゴー!!WGP). 게임 「ミニ四駆 レッツ&ゴー!! POWER WGP2」 한글 패치용.
출처: 나무위키 「등장인물」·「등장 머신」(assets/translation_guide/*.mhtml)에서 갈무리 + 원어 확인.

## 명명 정책 (확정)

- **1차 버전 = 일본 원어 음차 기준** (세이바 고 등). 아래 표의 **「원어 음차」 열이 실제 적용값**.
- 한국 방영판(「우리는 챔피언」) 로컬명(남궁 호 등)은 **병기·학습만** → **차후 별도 "한국명 버전"** 제작용.
- **MAX(애니 3기)·Return Racers 계열은 이 게임(WGP2) 범위 밖 → 미사용**(§7 참고용).
- 머신명은 원작·방영판 모두 가타카나 음차 유지 → 정책 무관 **음차 통일**.

### 호칭 규칙 (중요)

- **원문이 이름만 부르는 호격은 그 형태를 그대로 살린다.** 성을 임의로 붙이지 말 것.
  - `ゴー！` / `豪！` → **「고!」** (❌ 「세이바 고!」 아님)
  - `レツ！` / `烈！` → **「레츠!」**, `リョウ！` → **「료!」**, `トウキチ！` → **「토우키치!」**
- 즉 대화 중 부르는 톤·친밀도를 보존 — 풀네임(세이바 고)은 **원문에 풀네임이 나올 때만** 사용.
- 별명·애칭도 동일 원칙: 원문이 애칭이면 애칭 그대로(예: `おおがみ` 없이 부르면 그대로).

### 인격교환 에피소드 화자 라벨 규칙 (0xDE–0xE1)

`人格入れかえモード`로 인격이 뒤바뀐 구간. 원문 화자 라벨이 **`AなB` = 겉모습 A · 인격(정체) B** 구조다.

| 원문 라벨 | **적용** | 검증 근거 |
|---|---|---|
| `チイコなJ` | **「치이코가 된 제이」** | 정체=J(`ボク`, 0xE0 「ボクがJなんだよ」), 겉모습=치이코 |
| `JなチイコC` | **「제이가 된 치이코」** | 정체=치이코(`~ですわ♥`, 0xDF 「チイコですわ〜♥」). 레츠가 J 몸을 보고 「Jくん…じゃなかったチイコちゃん！」 |
| `藤吉な豪` | **「토우키치가 된 고」** | 정체=고(`オレ`, 0xE1 「オレだよ　豪だよ」) |
| `J？` `藤吉？` `豪？` | **「제이？」「토우키치？」「고？」** | `？`=주위가 정체를 아직 못 알아본 상태 표시 → 그대로 보존 |

- 규칙: **`AなB` → 「A가 된 B」**(겉모습이 A로 변한 B라는 뜻이 아니라 *A의 몸에 든 B*지만, 한국어로는 이 어순이 자연스럽고 정체가 뒤에 와 원문과 강세가 같다).
- 이 씬은 **어미가 정체를 드러내는 게 개그의 핵심** → 몸이 아니라 **인격 쪽 말투**를 반드시 따를 것(§1.5).

카운트는 공통: **글자(한글·라틴·문장부호)=1, 반각 띄어쓰기( )=0.5, 전각공백(　)=1.**

| 대상 | 상한 | 근거 |
|---|---|---|
| **673 정적 대사**(`dialogue.json` — c7_race·d0_story·c1_ui) | **≤ 13** | 초상화 있는 **좁은 레이스 상자** 기준. Mesen 실기 확인(2026-07-12). 넘으면 대화창 밖으로 밀려남(예: "영광의 골인까지 얼마 안 남았다！"=16자 → 넘침) |

- 빌드 가드: `scripts/build_patch.py`가 673에서 >13 줄을 발견하면 빌드 실패. **어드벤처 ≤16 자동 가드는 아직 없음 → 수동 확인**.
