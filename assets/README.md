# 자산 디렉터리

게임에 재삽입되는 정본 데이터와 사람이 편집하는 작업 이미지를 기능별로 나눈다.

| 경로 | 역할 |
|---|---|
| `fonts/body/` | 16×16 본문 대사용 TTF·글리프 맵·프리뷰 |
| `fonts/small/` | 8×8 메뉴·엔딩용 글꼴·글리프 맵 |
| `graphics/` | 화면·서브시스템별 승인 이미지, 번역표, 재편집 템플릿 |
| `screenshots/readme/` | 저장소 안내에 직접 사용하는 선별 스크린샷 |
| `translations/` | 문자열·주소·포인터·삽입문을 담은 기계 판독 SSOT |
| `translation_guide/` | 용어집·글리프표·번역 정책 |

새 그래픽은 화면의 소비 주체를 기준으로 `graphics/<subsystem>/`에 둔다. ROM에서 재생성되는
덤프와 개인 편집 파일은 커밋하지 않는다. 새 문자열이나 주소는 문서에만 적지 말고 먼저
`translations/`의 해당 JSON에 반영한다.
