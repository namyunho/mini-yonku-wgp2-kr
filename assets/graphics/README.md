# 그래픽 자산

각 디렉터리는 하나의 게임 화면 또는 재삽입 파이프라인에 대응한다.

| 경로 | 내용 |
|---|---|
| `intermission_logo/` | VICTORYS 에피소드 인터미션 로고 |
| `formation/` | 경기 전 포메이션 메뉴 라벨 |
| `garage/` | 개러지 부품 분류 |
| `machine_stats/` | 이지 세팅 기기 능력치 |
| `race_hud/` | 경기 중 DAMAGE/폭주 HUD |
| `result/courses/` | Result 경기장명 |
| `result/names/` | Result 선수명 |
| `stage_titles/` | 챕터 인트로 제목 10종 |
| `title_credits/` | 타이틀·크레딧 원본 이미지와 번역 BMP |

`*_workshop_256px.png`는 편집 템플릿, `*_workshop_approved.png` 또는 명시된 BMP는 빌드
승인본이다. `crops/`는 주소별 작업 단위다. `title_credits/extract/`는 ROM에서 다시 만들 수
있는 덤프라서 커밋하지 않는다.
